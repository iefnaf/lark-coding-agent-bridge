# Agent Instructions

## Agent skills

### Issue tracker

Issues are tracked in GitHub Issues; external PRs are not treated as a triage request surface. See `docs/agents/issue-tracker.md`.

### Triage labels

Triage uses the canonical label vocabulary: `needs-triage`, `needs-info`, `ready-for-agent`, `ready-for-human`, `wontfix`. See `docs/agents/triage-labels.md`.

### Domain docs

Domain docs use a single-context layout: root `CONTEXT.md` plus root `docs/adr/`. See `docs/agents/domain.md`.
